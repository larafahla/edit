# 6. Index an object

Write a function that given a list of objects, returns an object with all elements indexed by the property id.

From this:

```js
[
    {
        name: 'teste1',
        id: 1
    },
    {
        name: 'teste 2',
        id: 2
    }
]
```

To this:

```js
{
    1: {
        name: 'teste1',
        id: 1
    },
    2: {
        name: 'teste2',
        id: 2
    }
}
```
